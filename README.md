App to manage game matches.

## Features

- Login com Discord via OAuth2
- Schedule matches
- List matches
- View match details
- For server owners:
    - Share invitation link
    - Enter Discord server

## Tech Stack

- React Native
- TypeScript
- OAuth2
- Expo

## Run Locally

Clone project

```bash
  git clone https://github.com/lorenasg1/nlw-gameplay.git
```

Enter directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm run

  or

  yarn
```

Start the server

```bash
  expo start
```

Install Expo Go on your device via AppStore on PlayStore;
Scan the QRCode;
